# Streakr
Streakr is a gamified platform which develops a healthy addiction of fitness, and productivity for users by employing features such as streaks, levels, shops, in-app currencies, plans, challenges, etc. Streakr emphasises on improving health and productivity by utilising similar psychological mechanisms to the ones used by the most addicting platforms in the world. With minimal costs and fees, Streakr aims to be the change the world needs, using our weakness and addictions to grow.

# How to use Streakr
As Streakr prioritises a no friction experience, users may begin using streakr without an account, or an account system. All your data is stored in your local storage (Don't worry, we dont have any of your sensitive information!). To get started, users must click the 'Start Workout' button. That's it! Our humongous database will provide you with a list of exercises to perform, along with the instructions for them.

# Features
Streakr utilises an xp-based level system to keep you "hooked up" to fitness. Other features such as streakes keep users returning for more!
